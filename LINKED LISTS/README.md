### 3Ô∏è‚É£ Find the Middle of a Linked List
**Problem:**  
Identify the middle node of a linked list.  
(If even length, return the second middle node.)

**Approach:**  
- Use slow and fast pointers

**Time Complexity:** O(n)  
**Space Complexity:** O(1)

**Approach:**

**Time Complexity:** O(n + m)  
**Space Complexity:** O(1)

**Space Complexity:** O(1)
