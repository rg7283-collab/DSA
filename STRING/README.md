### 2Ô∏è‚É£ Valid Palindrome
**Problem:**  
Determine whether a string is a palindrome, considering **only alphanumeric characters** and **ignoring case**.

**Approach:**  
- Use two pointers

**Time Complexity:** O(n)  
**Space Complexity:** O(1)

### 3Ô∏è‚É£ Valid Anagram
**Problem:**

**Approach:**

**Time Complexity:** O(n)  
**Space Complexity:** O(1)

### 4Ô∏è‚É£ Longest Common Prefix
**Problem:**

**Approach:**  
- Use vertical scanning

**Time Complexity:** O(n √ó m)  
**Space Complexity:** O(1)  

(`n` = number of strings, `m` = length of shortest string)

### 5Ô∏è‚É£ Count Vowels in a String
**Problem:**

**Approach:**

**Time Complexity:** O(n)  
**Space Complexity:** O(1)

---

### 6Ô∏è‚É£ Remove Duplicate Characters from String

**Approach:**

**Space Complexity:** O(1)
